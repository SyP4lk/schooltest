# Блиц-тест по теме конфликта поколений

Готовый статический мини-сайт для GitHub Pages.

## Состав
- `index.html` - структура страницы
- `styles.css` - оформление
- `script.js` - логика теста и подсчёт результата

## Как выложить на GitHub Pages
1. Создайте новый репозиторий на GitHub.
2. Загрузите в него все файлы из этой папки.
3. Откройте `Settings -> Pages`.
4. В разделе Build and deployment выберите:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
5. Сохраните настройки и дождитесь публикации.
6. После этого GitHub Pages выдаст публичную ссылку на сайт.

## QR-код
После публикации скопируйте ссылку на сайт и создайте QR-код в любом генераторе QR. Потом этот QR можно вставить в презентацию или на отдельный лист к защите.
